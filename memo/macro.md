# マクロ

## 2026-02-14

### `println!` の `!` はマクロ呼び出しの印

- Rustで `!` がついた呼び出し（`println!`, `vec!`, `format!` など）は**マクロ**
- `!` を外すと関数や変数として探されるので「not found in this scope」エラーになる
- マクロ = **コンパイル前にコードを展開（生成）する仕組み**。「コードを書くコード」
- 関数との違い:
  - 関数は実行時に動く、マクロはコンパイル前に展開される
  - 関数は引数の数が固定、マクロは可変長OK
- `println!` が import なしで使えるのは **prelude**（自動 import）のおかげ
